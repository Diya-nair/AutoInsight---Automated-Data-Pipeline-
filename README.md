# AutoInsight---Automated-Data-Pipeline-
Automated data pipeline for data ingestion, processing, and visualization

AutoInsight is an end-to-end data pipeline project that automates the process of data ingestion, transformation, analysis, and visualization. The system takes raw data (CSV), processes it using Python, and provides meaningful insights through REST APIs and an interactive frontend dashboard.
This project demonstrates practical concepts of data engineering, backend development, and frontend integration.

Key Features
* Data ingestion from CSV files
* Data cleaning and preprocessing using Pandas
* REST API development using FastAPI
* Customer search functionality (/search_customer)
* Country-wise analytics (/top_countries)
* Summary statistics (/summary)
* Interactive frontend with search, sort, and filter


Tech Stack
* Programming Language: Python
* Libraries: Pandas
* Backend Framework: FastAPI
* Frontend: HTML, CSS, JavaScript

Project Workflow
* Load raw data from CSV
* Clean and preprocess using Pandas
* Serve processed data through APIs
* Fetch API data in frontend
* Display data using tables and charts
